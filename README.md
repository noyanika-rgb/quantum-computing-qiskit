# quantum-computing-qiskit
This repository contains all the experiments performed of  Quantum Computing  using Qiskit.  The experiments cover fundamental concepts such as qubits, quantum gates, superposition, entanglement, and basic quantum algorithms.

##  Technologies Used

##  List of Experiments

### 1. Introduction to Quantum Computing and Qiskit

Understanding basic concepts and quantum programming framework.

### 2. Installation of Qiskit and Environment Setup

Setting up Python environment and verifying Qiskit installation.

### 3. Quantum Bits (Qubits) and Quantum Registers

Understanding qubits, quantum registers, and classical registers.

### 4. Creating a Simple Quantum Circuit

Designing and simulating basic circuits.

### 5. Hello Quantum World Program

Executing a simple quantum program.

### 6. Single-Qubit Gates

Implementation of Pauli-X, Pauli-Y, and Pauli-Z gates.

### 7. Hadamard Gate and Bloch Sphere

Creating superposition and visualizing qubit states.

### 8. Quantum Superposition

Generating superposition states using Hadamard gate.

### 9. Quantum Measurement and Probability

Analyzing probabilistic outcomes of qubits.

### 10. Quantum Entanglement

Understanding correlation between qubits.

### 11. Bell States

Implementation using Hadamard and CNOT gates.

### 12. Introduction to Quantum Algorithms

Understanding constant and balanced functions.

### 13. Deutsch–Jozsa Algorithm (Implementation)

Designing and executing the algorithm.

### 14. Deutsch–Jozsa Algorithm (Analysis & Mini Project)

Analyzing results and demonstrating quantum advantage.

##  How to Run Experiments

1. Install dependencies:

```
pip install qiskit qiskit-aer matplotlib
```

2. Run any experiment:

```
python exp1.py
python exp2.py
...
```
